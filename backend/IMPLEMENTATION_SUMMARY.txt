## WORKFLOW

### User Uploads File:
1. User sends LAS/LAZ file to POST /api/lidar/datasets/upload/
2. File saved to MEDIA_ROOT/lidar_uploads/
3. LidarDataset record created with status='uploaded'
4. Django signal automatically triggered → processor.extract_features()
5. All metrics calculated
6. ForestCompartment records created with metrics
7. Status updated to 'completed' ✓

### No Manual Processing Needed!
The entire process is automatic via Django signals.

## COMMANDS TO USE

Process unprocessed datasets:
  python manage.py process_lidar_datasets

Reprocess all datasets:
  python manage.py process_lidar_datasets --all

Process specific dataset:
  python manage.py process_lidar_datasets --dataset-id=5

Run tests:
  python manage.py test lidar_app -v 2

Start development server:
  python manage.py runserver
